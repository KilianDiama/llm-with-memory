# 🧠 Cognitive Monolith

A fully modular, deterministic, and invariant‑driven cognitive architecture combining:

- Vector‑based memory with multi‑criteria retrieval  
- Semantic graph with normalized relations  
- Fractal abstraction engine  
- Contradiction detection  
- Executive system with goals  
- Policy engine for cognitive style control  
- Unified LLM interface  
- Full cognitive loop with self‑evaluation  

This repository provides a **clean, production‑grade reference** for building advanced cognitive agents with persistent memory, semantic reasoning, and metacognitive capabilities.

---

## 🚀 Features

### **1. Memory Engine (Episodic, Semantic, Meta, System)**
- Normalized float32 embeddings  
- Energy‑based decay  
- Recency, frequency, emotional weighting  
- Short‑term buffer  
- SQLite persistence  
- Deterministic invariants on every update  

### **2. Semantic Graph**
- Unique node IDs  
- Normalized outgoing weights  
- Spreading activation with configurable decay  
- Concept nodes with tags and content  

### **3. Fractal Abstraction Engine**
- High‑level synthesis of recent concepts  
- JSON‑validated output  
- Automatic creation of meta‑concept nodes  
- Connects abstractions back into the graph  

### **4. Contradiction Engine**
- Lightweight negation‑based contradiction detection  
- Works on top‑k retrieved memories  

### **5. Executive System**
- Goal management  
- Priority‑based scheduling  
- Integrated into the cognitive loop  

### **6. Policy Engine**
- Controls temperature, style, prudence  
- Driven by self‑evaluation metrics  

### **7. LLM Engine**
- Single invariant entry point for all LLM calls  
- Supports JSON‑validated outputs  
- Timeout‑safe with fallback messages  

### **8. Full Cognitive Loop**
- Memory tick + decay  
- Episodic encoding of user input  
- Context building (memory + graph + goals + contradictions)  
- Draft generation  
- Self‑evaluation  
- Final refinement  
- Optional fractal abstraction every N steps
